# Credit Card Fraud Detection Using Machine Learning

## Overview
This project detects fraudulent credit card transactions using Machine Learning algorithms.

The model is trained on highly imbalanced transaction data and predicts whether a transaction is fraudulent or valid.

---

## Features
- Data preprocessing
- Exploratory Data Analysis
- Correlation heatmap
- Random Forest Classifier
- SMOTE oversampling
- Model evaluation metrics
- Confusion Matrix Visualization

---
